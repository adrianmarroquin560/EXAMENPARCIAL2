Una lista doblemente enlazada circular es una estructura de datos donde cada nodo guarda un valor y dos referencias: una al nodo siguiente y otra al nodo anterior. A diferencia de una lista normal en la versión circular el último nodo apunta de regreso al primero y el primero apunta al último asi formando un ciclo cerrado asi esto  permite recorrer la lista en ambas direcciones sin llegar al final.

Enlace del video: https://youtu.be/FynyljfSSJ4

   LISTA DOBLEMENTE ENLAZADA CIRCULAR

4. Eliminar al final   
5. Buscar elemento
